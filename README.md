# This is an implementation package of styper, a static type inference tool for Python

## uncompress data files

```bash
unzip RQ1.zip
unzip RQ2.zip
unzip RQ3.zip
```

## obtain numbers and results

```bash
python rq1.py
sh rq2.sh
python rq3.py
```

## how to execute our tool.

```
python typeinfer.py <entry point>
```
